# Changelog

本项目从 1.3.0 开始遵循语义化版本，并使用统一的 `vX.Y.Z` Git 标签。

## [Unreleased]

## [1.3.1] - 2026-07-13

### Fixed

- 冻结应用目录测试改用平台原生绝对路径，修复 Linux CI 将 Windows 盘符路径解析为相对路径的问题。
- 二阶浅层最优性正确性测试不再使用易受覆盖率插桩影响的单次性能超时，避免慢速 CI runner 误报失败。

### Changed

- README 重写为逐步操作手册，补充便携版、源码环境、六面拍摄、CI 跳过项和标签发布的完整流程。

## [1.3.0] - 2026-07-13

### Added

- Windows x64 便携包构建、磁盘预检、基础 PDB 与快速求解表随包发布。
- Python 3.10–3.14 锁定依赖、统一开发环境和完整检查入口。
- 受限 Windows 环境下的并行搜索自动单进程回退。
- 单一版本源、版本一致性门禁和标签驱动的 GitHub Release 流程。

### Changed

- CI 扩展到 Python 3.10–3.14，并发布经过测试的 Windows ZIP Artifact。
- 前端显示、HTTP 响应头、API、Python 包元数据和发布文件名共享同一版本。

## [1.2.0] - 2026-07-13

- 改进真实光照下的颜色距离和红橙判别。
- 优化深度 18/19 的原生最短解搜索、Tail-6 v4 和正反向预选。
- 建立 Python、前端和 Windows 原生求解器 CI 基线。
