魔方拍照最短解 - Windows 便携版
================================

使用方法
1. 完整解压 ZIP，不要直接在压缩包中运行。
2. 双击“启动魔方求解器.cmd”。
3. 浏览器会自动打开本地页面；如未打开，请使用窗口中显示的 http://127.0.0.1:端口/。
4. 使用结束后，在运行窗口中按 Ctrl+C 停止服务。

说明
- 应用只监听本机 127.0.0.1，不会把照片上传到外部服务器。
- 包内已包含 Python 运行环境、OpenCV、前台快速求解表、C++ 求解器和严格最短证明所需的基础 PDB。
- 首次二阶求解会在内存中生成二阶剪枝表，之后在本次运行中复用。
- 随包基础 PDB 可完成严格最短证明；包含可选 Tail-6 PDB 的扩展包通常能更快处理深层状态。
